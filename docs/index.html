OGRS3

Calculate Offender Group Reconviction Scale v3

POST /calculate/ogrs3

calculate OGRS3 Rating on a known set of parameters

OGRS3CalculatorRequest

Details of a subject to calculate OGRS3 Rating on

Request Example
{
  "gender": "M",
  "birthDate": "1989-04-22T00:00:00.000Z",
  "convictionDate": "2013-02-20T00:00:00.000Z",
  "firstSanctionDate": "2001-09-10T00:00:00.000Z",
  "previousSanctions": 25,
  "assessmentDate": "2013-03-31T00:00:00.000Z",
  "currentOffenceType": 12
}

Schema Definitions

OGRS3CalculatorRequest: object

gender: string , x ∈ { M , F }

Subjects gender

birthDate: string (DateTime)

Subjects data of birth

convictionDate: string (DateTime)

for sentenced offenders, this is the age at sentence. For PSRs, it is the offender's current age

firstSanctionDate: string (DateTime)

date the subject was first sanctioned

previousSanctions: number

a sanction is a caution, reprimand or final warning, or a court appearance resulting in conviction. A sanction can be for one or many offenses, provided they are all dealt with on the same day

assessmentDate: string (DateTime)

the date that the offender could be in a position to reoffend again (ignoring offenses committed whilst in custody)

currentOffenceType: number

the identifier of the current offense type taken from the OGRS3 current offense categories

Example
{
  "gender": "M",
  "birthDate": "1989-04-22T00:00:00.000Z",
  "convictionDate": "2013-02-20T00:00:00.000Z",
  "firstSanctionDate": "2001-09-10T00:00:00.000Z",
  "previousSanctions": 25,
  "assessmentDate": "2013-03-31T00:00:00.000Z",
  "currentOffenceType": 12
}

CustomisedOGRS3CalculatorRequest: object

gender: string , x ∈ { M , F }

Subjects gender

birthDate: string (DateTime)

Subjects data of birth

convictionDate: string (DateTime)

for sentenced offenders, this is the age at sentence. For PSRs, it is the offender's current age

firstSanctionDate: string (DateTime)

date the subject was first sanctioned

previousSanctions: number

a sanction is a caution, reprimand or final warning, or a court appearance resulting in conviction. A sanction can be for one or many offenses, provided they are all dealt with on the same day

assessmentDate: string (DateTime)

the date that the offender could be in a position to reoffend again (ignoring offenses committed whilst in custody)

currentOffenceFactor: number

the factor of the offense type to be used as the OGRS3 current offense categories

Example
{
  "gender": "M",

  "convictionDate": "2013-02-20T00:00:00.000Z",
  "firstSanctionDate": "2001-09-10T00:00:00.000Z",
  "previousSanctions": 25,
  "assessmentDate": "2013-03-31T00:00:00.000Z",
  "currentOffenceFactor": 0.6574
}

OGRS3RatingResult: object

calculatorVersion: string

The version of the calculator algorithm used to calculate the rating

OGRS3: object[]

the calculated first and second year OGRS3 risk

OGRS3PercentileRisk: object[]

the calculated first and second year OGRS3 percentile risk

Example
{
  "calculatorVersion": "1.2.3",
  "OGRS3": [
    0.0056783258
  ],
  "OGRS3PercentileRisk": [
    0.56
  ]
}

OGRS3Rating: number (double) , { x ∈ ℝ | 0 ≤ x ≤ 0.001 }

An Offender Group Reconviction Scale Rating

Example
0.0056783258

OGRS3PercentileRisk: number (double) , { x ∈ ℝ | 0 ≤ x ≤ 1 }

A percentile Offender Group Reconviction Scale Rating

Example
0.56
